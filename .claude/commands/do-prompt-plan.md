1. 開啟 **@prompt_plan.md** 並識別任何未標記為已完成的 prompts。
2. 對於每個未完成的 prompt：
   - 再次確認它是否真的未完成（如果不確定，請尋求澄清）。
   - 如果你確認它已經完成，跳過它。
   - 否則，按描述實作它。
   - 確保測試通過，程式能夠建置/執行
   - 用清楚的 commit message 將變更 commit 到你的 repository。
   - 更新 **@prompt_plan.md** 將此 prompt 標記為已完成。
3. 完成每個 prompt 後，暫停並等待使用者檢視或回饋。
4. 根據使用者指示重複下一個未完成的 prompt。
